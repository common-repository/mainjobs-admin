Custom admin enhacements for Mainjobs Webs in WordPress.

== Description ==
Makes some changes in WordPress admin for better customization in Mainjobs Webs:

- Custom logo and login screen.
- Unpublished posts in other colour.
- Custom info.


Made by [Closemarketing](https://close.marketing/)

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your
WordPress installation and then activate the Plugin from Plugins page.
